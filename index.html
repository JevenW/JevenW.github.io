科学上网：Vultr VPS 搭建 Shadowsocks

购置服务器

按下不表。

搭建 Shadowsocks

连接 SSH

使用命令：ssh root@server_ip

安装 BBR 加速

预备工作

CentOS：yum -y install wget

Debian/Ubuntu：apt install -y wget

开始安装

使用命令：
wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh && chmod +x bbr.sh && ./bbr.sh

查看是否安装成功

使用命令：lsmod | grep bbr
观察是否显示「tcp_bbr」字样，如有，则安装成功。

安装 Shadowsocks

依次运行下面三行命令。

下载脚本

使用命令：
wget — no-check-certificate -O shadowsocks.sh https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh

更改权限

使用命令：chmod +x shadowsocks.sh

运行脚本

使用命令：./shadowsocks.sh 2>&1 | tee shadowsocks.log

根据屏幕提示，依次输入 密码，端口，选择加密方式。

服务端搭建完成！

多用户配置

准备配置信息

创建配置文件

使用命令：vi /etc/shadowsocks.json
出现一列波浪线即进入 vim。按 a ，复制上面准备好的配置信息，再按 Esc ，输入 :wq 保存退出。

重启 Shadowsocks 生效

使用命令：/etc/init.d/shadowsocks restart

开启端口

配置完成后，多用户端口也许无法翻墙，这时需要去防火墙开启相应端口。

开启所有端口后，需要重启网络端口：firewall-cmd --reload
